# Calendrier prévisionnel — InnovQube (exemple)

## Phase 0 — Préparation (0–1 mois)
- Audit technique & conformité RGPD.
- Définition des KPIs pilote.
- Mise en place environnements de test (Apaleo, Swagger).

## Phase 1 — Pilotes & tests (1–4 mois)
- Intégrations API (Apaleo, autres PMS).
- Lancement pilote InnovRental (petit panel).
- Recueil feedback & corrections.

## Phase 2 — Pré-lancement commercial (4–6 mois)
- Améliorations produit (UX, automatisation).
- Plan marketing ciblé sur propriétaires indépendants.
- Mise en avant conformité RGPD & option souveraine.

## Phase 3 — Scale-up (6+ mois)
- Expansion marché France / UE.
- Partenariats (agences, offices du tourisme).
- Optimisation coût via FinOps & hébergement responsable.
