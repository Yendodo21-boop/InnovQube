# InnovQube
InnovQube est une PME positionnée sur : conseil en transformation digitale, développement de solutions SaaS (*InnovRental*, *InnovGuide*) et services à la personne (AW Service). Le marché des locations courte durée est dynamique et la demande pour des PMS/DG intégrés croît.
